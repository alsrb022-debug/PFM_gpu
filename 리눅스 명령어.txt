pwd : 현재 위치 확인
ls : 폴더 목록 보기 (-l : 자세히 보기, -a 숨김파일 보기, -la)
cd : 폴더이동 (cd .. : 상위 폴더 이동, cd ~ : 홈으로 이동)
mkdir name : name 폴더만듬
mkdir -p pfm/src : pfm폴더 만들고 하위에 src 만듬
touch name : 파일 만들기
nvim name : 파일편집
	i : 입력모드
	esc : 명령모드
	:w  : 저장
	:q : 종료
	:wq 저장 후 종료
cat name :  파일 보기
less name : 긴 파일 보기
cp 원본 복사본 : 파일 복사
cp -r folder newfolder : 폴더 복사
mv name newname : 이름 변경
rm 파일 : 삭제
rm -r 폴더
